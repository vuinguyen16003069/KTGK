# 🌟 Dự án KTGK JavaScript

## 📝 Mô tả

Dự án này là bài kiểm tra giữa kỳ môn JavaScript, tập trung vào việc xây dựng một ứng dụng web tương tác sử dụng HTML, CSS, JavaScript và bootstrap.

## 🛠️ Công nghệ sử dụng

- HTML5
- CSS3
- JavaScript (ES6+)
- Custom Fonts:
  - Inter (Medium, Regular)
  - Playfair Display (Bold)

## 📂 Cấu trúc dự án

```
KTGK/
├── assets/                 # Thư mục chứa tài nguyên
│   ├── fonts/             # Fonts tùy chỉnh
│   │   ├── Inter-Medium.woff2
│   │   ├── Inter-Regular.woff2
│   │   └── PlayfairDisplay-Bold.ttf
│   └── images/            # Hình ảnh (14 ảnh)
│       └── [1-14].png
├── src/                   # Mã nguồn
│   ├── script.js         # Logic JavaScript
│   └── styles.css        # Styles CSS
├── index.html            # File HTML chính
├── package.json          # Cấu hình dependencies
└── README.md             # Tài liệu dự án
```

## 🚀 Hướng dẫn cài đặt

1. Clone dự án:
   ```bash
   git clone [url-repository]
   ```
2. Di chuyển vào thư mục dự án:
   ```bash
   cd KTGK
   ```
3. Cài đặt dependencies:
   ```bash
   npm install
   ```
4. Chạy dự án:
   - Mở file `index.html` bằng trình duyệt web
   - Hoặc sử dụng Live Server trong VS Code
